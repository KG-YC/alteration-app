# 改衣記帳 App 使用說明

**網址**：https://kg-yc.github.io/alteration-app/

**登入**：用 Google 帳號登入（僅限授權帳號，未授權會顯示「此帳號未授權使用」）

App 下方有 4 個分頁：**建檔｜統計｜查歷史｜設定**

---

## 1. 建檔（新增訂單）

1. **日期**：預設今天，可改。
2. **客戶**：
   - 「選現有」：從下拉選單選已建檔的客戶
   - 「新客戶」：填聯絡帳號（IG/FB/LINE，必填且唯一）、姓名、電話、備註
3. **類型／師傅／來源平台**：下拉選擇，選單內容可在「設定」頁增減。
4. **修改項目**：一張單可填多項（品牌、改法、尺寸、收費、師傅工資、材料費、備註各自獨立記錄）。
5. 下方會即時顯示：**本單收入 / 材料費合計 / 預估毛利**（已自動扣師傅工資、材料費、電費）。
6. 按「**建檔送出**」完成。

---

## 2. 統計

- **篩選期間**：選年份／月份（或全部）。
- 上方三個色塊：**待處理 / 完成待取 / 已結案** 訂單數，點擊可直接跳到「查歷史」看該狀態的訂單。
- 總覽數字：總收入、我的總毛利、師傅工資合計、材料費合計、電費合計、訂單數。
- **各項目毛利**：依修改項目（如「換拉鍊」「改褲長」）統計毛利。
- **每月毛利 / 每年毛利 / 各師傅工資 / 各平台收入**：每一列都可以點擊，會跳到該期間/該師傅/該平台的**訂單明細清單**，再點單筆訂單可看完整內容。

---

## 3. 查歷史

- **狀態篩選**：全部 / 待處理 / 完成待取 / 已結案。
- **維度搜尋**：可勾選「人名」「帳號」「修改項目」，再輸入關鍵字搜尋（也可不勾選，直接模糊搜尋全部）。
- **年月快選**：選年份/月份快速篩選。
- **自訂日期範圍**：指定起訖日期。
- 點擊單筆訂單進入明細，可：
  - **編輯**：修改日期、類型、師傅、平台、各修改項目、工資、電費、備註，存檔後自動重算金額。
  - **刪除**：需二次確認。
  - **變更狀態**：待處理 → 完成待取 → 已結案。

---

## 4. 設定

### 下拉選單管理

管理「項目名稱／品牌／改法／師傅／類型／來源平台」這幾個欄位的選項：

- 點 X 刪除選項（不影響已存在的訂單紀錄）
- 輸入文字後新增選項

這些選項會套用到「建檔」與「編輯訂單」的下拉選單。

### 客戶管理

列出所有已建檔客戶（顯示姓名、帳號 @id、累計訂單數）。

- 點垃圾桶圖示 → 二次確認 → 刪除該客戶
- **刪除客戶不會刪除其歷史訂單**：訂單仍會保留，只是不再顯示客戶姓名（顯示空白姓名 + @id）
